How to install

Arduinoを起動して、
スケッチ -> ライブラリのインクルード -> .zip形式のライブラリをインストール…
の順番でAdafruit_NeoPixel-master.zipをインストールする。

Sampleはスイッチサイエンスのブログのソースコード。
Keyboardに使うのはmain
